# 📡 Wireless Network with Internet Connectivity (Home/Office Network Design)

## 📌 Overview
This project demonstrates the design and configuration of a **wireless network with internet connectivity** using Cisco Packet Tracer. The network simulates a real-world **home or small office setup**, integrating wired and wireless devices with an ISP connection.

The topology includes a wireless router, access point, and multiple devices connected through both **Ethernet and WiFi**.

---

## 🏗 Network Architecture

- **Wireless Router (Home Router-PT-AC)** acting as the central device  
- **ISP Cloud** providing internet connectivity  
- **Switch** connecting wired devices  
- **Access Point** extending wireless coverage  
- **Laptops and PCs** connected via wired and wireless networks  

---

## ⚙️ Key Features

- Wireless network configuration (WiFi)
- Integration with ISP for internet access
- Wired and wireless device connectivity
- Use of an access point to extend network coverage
- Real-world home/office network simulation
