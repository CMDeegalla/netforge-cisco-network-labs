# 🌐 3-Tier Campus Network Design (RIP & Aggregated Links)

## 📌 Overview
This project demonstrates a scalable **3-tier campus network architecture** using Cisco Packet Tracer. The design follows industry standards with **Core, Distribution, and Access layers** to ensure high performance, scalability, and structured network management.

The network uses **RIP (Routing Information Protocol)** for dynamic routing between multiple network segments.

---

## 🏗 Network Architecture

### 🔹 Core Layer
- Central backbone of the network
- Connects all distribution switches
- Ensures high-speed data transfer

### 🔹 Distribution Layer
- Handles routing and policy control
- Connects the access layer to the core

### 🔹 Access Layer
- Provides connectivity to end devices
- Uses Layer 2 switches

---

## ⚙️ Key Features

- 3-tier hierarchical network design
- RIP dynamic routing implementation
- Structured IP addressing and subnetting
- End-to-end connectivity across all layers
- Scalable and modular design

---

## 🧠 Technologies Used

- Cisco Packet Tracer
- Layer 2 & Layer 3 Switching
- RIP Routing Protocol
- Subnetting & IP Addressing

---

## 📊 Network Details

- **1 Core Switch (Layer 3)**
- **3 Distribution Switches**
- **3 Access Switches**
- **3 End Devices (PCs)**
- Multi-layer hierarchical topology
