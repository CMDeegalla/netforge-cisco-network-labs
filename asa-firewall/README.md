# 🔐 Cisco ASA Firewall Configuration (Basic Security Lab)

## 📌 Overview
This project demonstrates the configuration of a **Cisco ASA 5505 firewall** to control and secure network traffic between internal and external networks.

The lab simulates a real-world scenario in which a firewall enforces security policies, access control, and traffic filtering.

---

## 🏗 Network Architecture

- **Cisco ASA 5505 Firewall** acting as the security gateway  
- **Router (2911)** representing external/public network  
- **Switch (Layer 2)** connecting internal devices  
- Multiple PCs representing internal users  

---

## ⚙️ Key Features

- Firewall configuration using **Cisco ASA**
- Implementation of **Access Control Lists (ACLs)**
- Traffic filtering between internal and external networks
- Controlled communication based on security rules
- Simulation of a real-world network security setup

---

## 🧠 Technologies Used

- Cisco Packet Tracer  
- Cisco ASA Firewall (5505)  
- ACL (Access Control Lists)  
- Routing & Switching basics  

---

## 📊 Network Details

- **1 ASA Firewall**
- **1 Router**
- **1 Switch**
- **3+ End Devices (PCs)**
- Internal and external network segmentation
