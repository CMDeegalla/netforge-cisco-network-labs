# 🌐 Cyber-Resilient Campus Network (3-Tier VLAN & VoIP Design)

## 📌 Overview
This project demonstrates a **cyber-resilient campus network architecture** implemented using Cisco Packet Tracer. The design integrates **VLAN segmentation, VoIP communication, and a 3-tier network model (Core, Distribution, Access)** to simulate a real-world enterprise environment.

The network ensures secure, scalable, and efficient communication among different user groups, including students, staff, and voice services.

---

## 🏗 Network Architecture

### 🔹 Core Layer
- Central backbone switch connecting all distribution switches  
- Provides high-speed and reliable data forwarding  

### 🔹 Distribution Layer
- Handles routing and inter-VLAN communication  
- Connects core and access layers  

### 🔹 Access Layer
- Provides connectivity to end devices (PCs, IP Phones, Servers)  
- Uses Layer 2 switches  

---

## ⚙️ Key Features

- 3-tier hierarchical network design  
- VLAN segmentation (Student, Staff, Voice, Systems)  
- Inter-VLAN routing for communication between departments  
- VoIP integration using IP phones and a VoIP server  
- Server integration for system services  
- Scalable and fault-tolerant network structure  

---

## 🧠 Technologies Used

- Cisco Packet Tracer  
- VLAN Configuration  
- Inter-VLAN Routing  
- VoIP (IP Phones & VoIP Server)  
- Layer 2 & Layer 3 Switching  
- Network Design & Segmentation  

---

## 📊 Network Details

- **1 Core Switch (Layer 3)**
- **1 Distribution Switch**
- **Multiple Access Switches**
- **10+ End Devices (PCs & IP Phones)**
- **VoIP Server + System Server**
- Multiple VLANs for segmentation
